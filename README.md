# SQL Retail Sales Analysis

## Overview

This project analyzes retail sales data using SQL to answer real-world business questions. The goal is to perform data cleaning, exploratory analysis, and generate actionable insights from sales transactions.

The project covers customer behavior, sales performance, category analysis, and time-based trends using SQL queries.

---

## Dataset Information

The dataset contains retail transaction records with the following attributes:

* Transaction ID
* Sale Date
* Sale Time
* Customer ID
* Gender
* Age
* Product Category
* Quantity
* Price Per Unit
* Cost of Goods Sold (COGS)
* Total Sale

---

## Project Workflow

### 1. Database Creation

* Created a retail_sales table
* Defined appropriate data types
* Loaded transaction data

### 2. Data Cleaning

* Checked for missing values
* Removed records containing NULL values
* Verified dataset consistency

### 3. Exploratory Data Analysis

Performed analysis on:

* Total sales records
* Total customers
* Product categories
* Sales transactions

### 4. Business Analysis

Answered key business questions such as:

* Sales on a specific date
* Clothing sales during November 2022
* Total sales by category
* Average customer age for Beauty products
* High-value transactions
* Transactions by gender and category
* Best-performing month each year
* Top 5 customers by sales
* Unique customers per category
* Sales distribution by shift (Morning, Afternoon, Evening)
